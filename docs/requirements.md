# 高木同学小豆岛圣地巡礼网站需求文档

## 项目目标

制作一个中文动画圣地巡礼攻略站，围绕小豆岛地点、路线和拍照建议，先发布可用 MVP，再逐步扩展真实地图、截图对比和变现内容。

## 第一版页面

- 首页：展示标题「高木同学小豆岛圣地巡礼地图」、副标题「一起走进动画里的海边小镇」和「开始巡礼」按钮。
- 圣地地图页：用静态卡片展示地点，不接真实地图 API。
- 打卡点详情页：展示地点简介、拍照时间、标签、停留时间和提示。
- 路线推荐页：展示半日、一日、两日路线。
- 关于页：说明项目定位、MVP 重点和后续方向。

## MVP 方案

- 技术栈：Next.js App Router + Tailwind CSS。
- 数据来源：本地 JSON 文件，方便后续手动维护。
- 视觉方向：日系、清爽、海边、青春感。
- 内容重点：6 个示例地点和 3 条路线。

## 实施步骤

1. 初始化 Next.js + Tailwind CSS 项目。
2. 新增本地数据文件 `src/data/spots.json` 和 `src/data/routes.json`。
3. 搭建公共导航、页脚、地点卡片和页面容器组件。
4. 实现首页、地图页、详情页、路线页、关于页。
5. 调整全局中文字体、配色和响应式布局。
6. 运行 lint/build/dev server 验证项目可运行。

## 后续变现思路

- 做「小豆岛圣地巡礼 PDF 攻略」低价数字产品。
- 接入酒店、交通、旅行保险或 eSIM 联盟链接。
- 发布 B 站、小红书、YouTube Shorts 巡礼视频，把网站作为资料页。
- 做截图对比、拍照机位、路线下载等高级内容。
- 为其他动画圣地复制同一套模板，形成主题站矩阵。
